# ROSMAP DLPFC Metabolomics

## Summary

Brain metabolomics data from the Religious Orders Study and Memory and Aging Project (ROSMAP), profiling ~600 metabolites in dorsolateral prefrontal cortex (DLPFC) samples.

## Dataset Information

- **Cohort**: ROSMAP (Religious Orders Study / Memory and Aging Project)
- **Brain region**: Dorsolateral prefrontal cortex (DLPFC)
- **Data type**: Untargeted metabolomics
- **Metabolites**: ~600 metabolites
- **Platform**: Mass spectrometry

## Omics Data

### Metabolite abundance matrices

Metabolite abundance data available through Synapse.
