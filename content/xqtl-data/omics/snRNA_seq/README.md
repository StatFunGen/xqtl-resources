# Single-Nucleus RNA-seq Data

Raw single-nucleus RNA-seq (snRNA-seq) datasets from ROSMAP brain tissue, processed separately by site (CUIMC and MIT) before integration. These datasets underlie the pseudo-bulk eQTL and ISSAC sQTL analyses. → See [expression data](../expression) and [splicing data](../splicing)
